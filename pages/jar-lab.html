Jar Lab

Low-Voltage Resonance Bench Setup in a Sealed Container

LAB MODULE · PRACTICAL WORKFLOW · REV 1.0

The Jar Lab is the practical, low-risk entry point for Resonant OS experiments. It translates theoretical stack concepts into a controlled bench-scale environment using low voltage, common materials, and repeatable observation routines.

Purpose

  • Create a stable container for humidity-managed resonance tests.
  • Observe stack response under small voltage bias and frequency nudges.
  • Record repeatable changes in potential, temperature drift, and signal noise floor.

Baseline Build

Minimum Lab Stack
Container
Glass jar with non-reactive lid and two feedthrough points.
Electrode Base
Copper primary output element with secure contact leads.
Driver Layer
Optional silver/palladium contact pair above copper.
Hydration Control
Small humidity source to keep environment stable and measurable.
Instrumentation
Low-voltage supply, multimeter, and simple data log sheet.

Operating Sequence

  1. Assemble dry and verify continuity before introducing humidity.
  2. Apply a small bias and wait for baseline stabilization.
  3. Step frequency or code pattern one change at a time.
  4. Log output deltas and reset between trials.
Related Tools: Use Resonance Calculator for pair values, Intent Programmer for sequence planning, and Stack Visualizer for layer planning.
⚠ Safety Notice

Use only low-voltage bench supplies and non-hazardous materials. Do not use radioactive, toxic, pressurized, or controlled substances. This page is conceptual and educational; all physical work must follow applicable lab and legal safety requirements.
